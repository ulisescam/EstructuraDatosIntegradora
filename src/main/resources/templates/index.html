Descripción del sistema

Esta aplicación web permite agregar, organizar y completar tareas personales o laborales. Internamente utiliza estructuras de datos básicas: una lista simplemente enlazada para el almacenamiento general, una pila para el historial de acciones y una cola para administrar las tareas pendientes en orden de llegada.
